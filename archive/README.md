# Archive

This folder contains prototype scripts that were used during the initial development phase to explore and test the MangaDex API.

## Contents

- **`javascript/`** - Node.js prototype scripts for testing API functionality
  - `answer.js` - CLI prototype for fetching random manga and checking answers
  - `search.js` - Simple search script
  - `scan.js` - Scanlation group info script
  - `stats.js` - Manga statistics script

- **`python/`** - Python prototype scripts (similar functionality)

## Status

**These files are deprecated and no longer used.** All functionality has been integrated into the main Next.js application in the `client/` folder.

These are kept for reference purposes only.
